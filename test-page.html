🎯 How to Test MrPlug

  1. Make sure the MrPlug extension is loaded in your browser
  2. Press Alt+Shift+F (Windows/Linux) or Cmd+Shift+F (Mac) to activate feedback mode
  3. Click on any element below to select it
  4. Describe your feedback (e.g., "This button is too small", "Wrong color", "Should be centered")
  5. See the AI analyze the element and suggest improvements!
